# Raven cover assets

Place the landing page album cover images in this folder using these names:

- `hero-album.jpg` - main MP3 player cover.
- `rapper-album.jpg` - list cover for a rapper album.
- `band-album.jpg` - list cover for a band album.
- `thriller.jpg` - Trending Albums cover for Thriller.
- `olivia-pretty-sad.jpg` - Trending Albums cover for You Seem Pretty Sad for a Girl So in Love.
- `bad-bunny-dtmf.jpg` - Trending Albums cover for Debi Tirar Mas Fotos.
- `showgirl.jpg` - Trending Albums cover for The Life of a Showgirl.
- `underscores.jpg` - Featured Artists photo for underscores.
- `beabadoobee.jpg` - Featured Artists photo for beabadoobee.
- `gab-ferreira.jpg` - Featured Artists photo for Gab Ferreira.

Use square images, ideally `1024x1024` or larger. JPG, PNG or WebP can work, but if you change the extension, update the paths in `hero-section.component.html`.
